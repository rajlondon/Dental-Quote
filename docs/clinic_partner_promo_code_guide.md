# Clinic Partner Guide: Promo Code System

## Introduction

This guide explains how your clinic can benefit from MyDentalFly's promo code system to attract more international patients and increase bookings.

## Types of Promotional Opportunities

### 1. Exclusive Clinic Packages

Your clinic can offer exclusive treatment packages with these benefits:
- **Guaranteed Visibility**: When a patient uses your package code, ONLY your clinic will be shown in search results
- **Bundled Treatments**: Create attractive packages combining multiple treatments
- **Special Pricing**: Offer package discounts that stand out from competitors
- **Additional Services**: Include value-added services like airport transfers and tourist activities

### 2. Standard Discount Codes

Your clinic can also participate in platform-wide discount promotions:
- **Percentage-based Discounts**: (e.g., 10%, 15%, 20% off)
- **Fixed Amount Discounts**: (e.g., £100, £200 off)
- **Seasonal Promotions**: Special offers tied to holidays or seasons

## How to Create a Package Promo Code

1. **Contact your MyDentalFly account manager** to request a new package promo code
2. **Specify the package details**:
   - Package name (e.g., "Premium Implant Package")
   - Included treatments and quantities
   - Regular price vs. package price
   - Additional services or benefits
   - Valid dates for the promotion
3. **Receive your unique promo code** (e.g., CLINICNAME2023)
4. **Promote your code** through your marketing channels

## Tracking Performance

1. **Monthly Reports**: You'll receive monthly reports showing:
   - Number of promo code uses
   - Conversion rate to actual bookings
   - Revenue generated from package codes
   
2. **Dashboard Access**: Access real-time statistics on your promo code performance through your clinic portal

## Best Practices

1. **Limited-Time Offers**: Create urgency by making packages available for a limited time
2. **Seasonal Promotions**: Align with off-peak seasons to attract patients during slower periods
3. **Tiered Package Structure**: Offer different package levels (Basic, Premium, Elite)
4. **Highlight Value-Added Services**: Emphasize services beyond dental treatments
5. **Cross-Promote**: Mention your package in all patient communications
